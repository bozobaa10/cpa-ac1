# แพลตฟอร์มฝึกทำข้อสอบ CPA – การบัญชี 1 (ปรนัย)

เว็บฝึกทำข้อสอบแบบเลือกตอบ (ช้อย) วิชาการบัญชี 1 รวม **364 ข้อ** จากข้อสอบ 20 ชุด (พ.ศ. 2561–2569)
เป็นเว็บแบบ static (ไฟล์ HTML ล้วน) เปิดได้ทุกอุปกรณ์ และโฮสต์ฟรีบน GitHub Pages ได้เลย

> ⚠️ **เพื่อการศึกษาเท่านั้น** โปรดตรวจสอบคำตอบกับมาตรฐานการบัญชีฉบับปัจจุบันอีกครั้งก่อนนำไปอ้างอิง

---

## ไฟล์ในโฟลเดอร์นี้

| ไฟล์ | หน้าที่ |
|------|---------|
| `index.html` | ตัวเว็บแอป (หน้าหลัก) |
| `data.js` | คลังข้อสอบ 364 ข้อ (เว็บโหลดไฟล์นี้) |
| `questions.json` | ข้อมูลข้อสอบชุดเดียวกัน แบบอ่านง่าย เผื่อนำไปใช้ต่อ |
| `.nojekyll` | ไฟล์ว่าง บอก GitHub ไม่ต้องประมวลผลพิเศษ (ห้ามลบ) |
| `README.md` | คู่มือนี้ |

> 📌 เวลานำขึ้น GitHub ให้อัปโหลด **ทุกไฟล์** โดยเฉพาะ `index.html`, `data.js`, และ `.nojekyll`

---

## วิธีนำขึ้นออนไลน์ฟรีด้วย GitHub Pages

ทำครั้งเดียวก็ใช้งานได้ตลอด ไม่ต้องเสียเงิน ทำตามทีละขั้นได้เลยครับ

### ขั้นที่ 1 — สมัคร / เข้าสู่ระบบ GitHub
ไปที่ [github.com](https://github.com) แล้วล็อกอิน (ถ้ายังไม่มีบัญชี กด **Sign up** สมัครฟรี)

### ขั้นที่ 2 — สร้าง Repository ใหม่
1. กดปุ่ม **+** มุมขวาบน → เลือก **New repository**
2. ตั้งชื่อ เช่น `cpa-ac1` (ตัวพิมพ์เล็ก ไม่เว้นวรรค)
3. เลือก **Public**
4. กด **Create repository**

### ขั้นที่ 3 — อัปโหลดไฟล์
1. ในหน้า repository กดลิงก์ **uploading an existing file** (หรือไปที่ **Add file → Upload files**)
2. ลาก **ทุกไฟล์** ในโฟลเดอร์นี้ลงไป (`index.html`, `data.js`, `questions.json`, `.nojekyll`, `README.md`)
3. กด **Commit changes** ด้านล่าง

> ถ้าลาก `.nojekyll` ไม่ได้ (บางเครื่องซ่อนไฟล์ขึ้นต้นด้วยจุด) ข้ามไปก่อนได้ เว็บยังทำงานได้ แค่ครั้งแรกอาจโหลดช้านิดหน่อย

### ขั้นที่ 4 — เปิดใช้งาน GitHub Pages
1. ในหน้า repository ไปที่แท็บ **Settings**
2. เมนูซ้าย เลือก **Pages**
3. หัวข้อ **Branch** เลือก **main** แล้วโฟลเดอร์ **/ (root)** → กด **Save**
4. รอประมาณ 1–2 นาที

### ขั้นที่ 5 — เปิดเว็บของคุณ
รีเฟรชหน้า **Settings → Pages** จะเห็นลิงก์เว็บขึ้นมา หน้าตาประมาณนี้:

```
https://<ชื่อบัญชีของคุณ>.github.io/cpa-ac1/
```

กดลิงก์นั้นได้เลย — เปิดได้จากมือถือ คอม แท็บเล็ต ทุกที่ และแชร์ให้เพื่อนได้

## การแก้ไข / เพิ่มข้อสอบภายหลัง
ข้อสอบทั้งหมดอยู่ในไฟล์ `data.js` (และสำเนาแบบอ่านง่ายใน `questions.json`)
แต่ละข้อมีโครงสร้างแบบนี้:

```js
{
  "id": "...",            // รหัสข้อ
  "exam": "AC1_69_01",    // ชุดข้อสอบต้นทาง
  "label": "2569/1",      // ปี/รอบ
  "topic": "TAS16 ที่ดิน อาคารและอุปกรณ์",
  "stem": "โจทย์คำถาม...",
  "options": { "a": "...", "b": "...", "c": "...", "d": "..." },
  "answer": "b",          // คำตอบที่ถูก
  "explanation": "เฉลย/คำอธิบาย (ถ้ามี)"
}
```

แก้ไขข้อความในไฟล์ แล้วอัปโหลดทับบน GitHub เว็บจะอัปเดตเองภายในไม่กี่นาที

## ฟีเจอร์ของเว็บ
- **2 โหมด:** ฝึกซ้อม (เฉลยทันทีทีละข้อ) / เสมือนสอบจริง (เฉลยตอนจบ)
- **เลือกชุดข้อสอบ** ตามปี/รอบ และ **เลือกหัวข้อ** (ตามมาตรฐาน TAS/TFRS) ได้
- เลือกจำนวนข้อ 10 / 20 / 40 / ทั้งหมด และสลับลำดับข้อได้
- รองรับคีย์ลัด (1–4 หรือ a–d เลือกคำตอบ, Enter ข้อถัดไป)
- หน้าสรุปผลคะแนน เกณฑ์ผ่าน 60% พร้อมตารางแยกตามหัวข้อ และทบทวนรายข้อ
- เก็บสถิติการทำข้อสอบไว้ในเครื่อง (localStorage)

## หมายเหตุข้อมูล
- รวบรวมจากข้อสอบ 20 ชุด (พ.ศ. 2561–2569) เฉพาะ**ข้อสอบปรนัย (ช้อย)** ส่วนข้อสอบอัตนัยยังไม่รวมในเวอร์ชันนี้
- ข้อที่ไม่มีเฉลยชัดเจน หรือมีเครื่องหมายคำตอบกำกวม ถูกคัดออกแล้วเพื่อความถูกต้อง
- ข้อสอบบางชุด (ปีเก่า) ผ่านการซ่อมการเข้ารหัสฟอนต์ภาษาไทย ตัวเลขทางบัญชีถูกต้องครบถ้วน แต่ข้อความบางจุดอาจมีการสลับตำแหน่งตัวอักษรเล็กน้อย
